Spela Warhammer 40k

1 / 7

Steg 1: Rörelsefas

Spelarna turas om att flytta sina modeller på spelbrädet. Varje modell har en specifik rörelsehastighet som bestämmer hur långt den kan flytta sig under rundan. Under denna fas kan spelarna också genomföra speciella rörelseaktioner eller taktiska manövrar.

2 / 7

Steg 2: Psykisk fas (om tillämpligt)

Om någon spelare har psykiska enheter eller förmågor, får de nu chansen att använda dem. Detta kan inkludera att kasta psykiska krafter för att skada fiendens enheter eller ge buffar till sina egna enheter.

3 / 7

Steg 3: Skjutfas

Spelarna turas om att utföra skjutningar med sina enheter. Varje enhet kan använda sina vapen för att skjuta på fiendens enheter inom deras räckvidd. Detta inkluderar att välja mål, rulla för att träffa, rulla för att såra och eventuellt rulla för att undvika träffar.

4 / 7

Steg 4: Stridsfas

Om det finns fiendliga enheter inom stridsräckvidd för någon spelares enheter, får de nu möjlighet att genomföra närstridsattacker. Detta inkluderar att välja mål, rulla för att träffa, rulla för att såra och eventuellt rulla för att undvika träffar.

5 / 7

Steg 5: Konsolideringsfas

Efter striden, om någon enhet fortfarande är i närstrid med en fiendeenhet, får spelaren möjlighet att konsolidera dessa enheter. Detta innebär vanligtvis att man flyttar enheterna för att dra nytta av deras position eller för att undvika att bli omringade.

6 / 7

Steg 6: Uppdatera effekter

Om det finns några pågående effekter som påverkar spelet, till exempel skador från tidigare strider eller buffar från tidigare runda, uppdateras dessa nu.

7 / 7

Steg 7: Avsluta runda

Rundan avslutas och den nästa rundan börjar med att spelarna återgår till steg 1 för att fortsätta spelet.
